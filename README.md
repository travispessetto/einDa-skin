Instruction for install (git required)
==========
* Download and execute installer
  *  ```wget https://git.io/vd3sX -O install```
  *  ```bash install```
* Enable skin
  * As reseller, go to the skin manager
  * Select einDa-skin
  * Click "Apply to me", and then the same with "Apply to all Users".
